‚öôÔ∏è Technical Information

Authentication

API menggunakan JWT (JSON Web Token) untuk autentikasi. Token dikirim melalui header Authorization: Bearer {token}

CORS

CORS dikonfigurasi untuk origin: localhost:5173-5177, localhost:3000, localhost:5000, dan 10.121.20.89:5173

File Upload

Upload menggunakan Cloudinary. Folders: absensi-presensi untuk foto check-in/out, absensi-laporan untuk bukti laporan

Time Restrictions

Check-in: 06:00 - 17:00 WIB
Check-out: Setelah 17:00 WIB
Timezone: Asia/Jakarta

Response Format

Semua response menggunakan format JSON dengan struktur:

message - Pesan status
data - Data response (jika ada)
error - Error message (jika error)
